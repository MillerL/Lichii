    家具产品在运输时应把纸箱按照箭头方向放置，堆放高度不能超过两层。

    新购置家具在拆包时应小心把封箱胶纸割开，注意不要割入太深，以免划破家具表面。开箱后小心拿出家具，找出安装图纸和配件。然后，仔细阅读安装图纸，严格按照图纸指示进行安装工作。安装完成后，请保留好图纸，以备需要拆卸和重新安装时使用。

    移动家具时，首先检查各处连接件的连接是否牢固。沙发类产品侧重检查沙发脚与框架的连接；柜体类产品检查柜体底脚与柜体的连接、门板间的连接、门把手的连接及内部零件的连接等。另外，柜体类产品应取出柜内及抽屉内物件，并将柜门和抽屉封紧，以免家具倾斜，部件滑落伤人。检查工作完成后，将整件家具产品小心抬离地面移动、轻放，切忌硬拖猛推，避免划伤地面或损坏家具底脚。

    1. 所有家具类产品应选择平稳的地面摆放，防止家具重心不稳引起的翻倒或变形。
    2. 沙发类家具产品摆放时应避免阳光直晒或靠近热源（空调热出风口和暖气附近）位置，阳光或热源长时间直接照在织物及皮革上，会破坏面料的组织，也会加速皮革的褪色。真皮沙发应放置在通风干燥处，避免潮湿、生霉和虫蛀。
    3. 木质家具产品也应避免阳光直晒或靠近热源，防止家具出现裂缝和局部褪色变色、抽屉抽拉不畅等现象。适宜家具的温度为18到24摄氏度，相对湿度为35%到40%。为了达到这个适宜环境，可以在室内摆放些绿植，养殖些花草或观赏鱼，还可以增添生活情趣。如使用空气加湿器，请勿将雾气直接喷在家具上。
    4. 柜体类家具内存放的物件，不能超过门的关闭位置，以防门面凸出变形。

    1. 柜体类产品的柜门在打开时不要施于重压，因为门铰是靠螺丝钉等固定在柜体上，起到了连接开合的作用。如果对门或门铰用力过大，会造成螺丝松动而使门铰脱落等问题出现。
    2. 家具产品不可直接触及酸、碱类的化学物品。
    3. 家具产品尤其是玻璃面产品在使用时应尽量避免被硬物、利器刮伤和划伤。
    4. 避免使用塑料制品保护木质桌面或与桌面直接接触，防止家具呼吸不畅而导致的面漆及木皮受损。
    5. 在家具上面放置高温物体时，一定要记得在家具表面放隔热垫，避免家具被烫伤。
    6. 切忌在沙发上蹦跳，以免线爆或塌陷变形。
    7. 沙发面套的拆装是技术含量较高的工作，客户如需拆装，建议您联系当地CAMERICH（锐驰）授权专卖店，安排专人来为您操作。如需自行拆装，请严格按照以下步骤进行，以防错误的拆装方法损坏面套，给您带来不必要的损失。
    A、拆面套：首先，将所有的坐垫、靠垫拿开，从沙发底部拉开面套的搭扣或拉链，用拉毛布（已配置）把搭扣粘上，以免面料挂住搭扣抽丝；然后，将面套由下往上缓缓地剥离，按照沙发的一边至另一边每次几厘米的步骤交替进行；最后，将面套完全从沙发上卸下。剥离过程切忌强拉硬拽，以免面套变形。
    B、拆垫子套：首先，拉开座垫或靠垫的拉链，用力挤压内胆使其变小，然后从开口处轻轻抽出内胆，将面套完全拆下。
    C、罩面套：首先要认清面套的正反面，从沙发最高处两侧开始套下，按照沙发的一边至另一边每次几厘米的步骤交替进行，特别注意是扶手及靠背的边角处要罩整齐，使布套贴紧沙发本体，抚平皱折；最后在沙发底部拉上拉链或按平搭扣。
    D、罩垫子套：首先要认清垫子套的正反面；打开垫子套的拉链后，应注意内胆斜度方向与垫子套一致；其次用力挤压内胆使其变小，从拉链口处塞进，使内胆与面套基本贴合；最后双手拍打垫子各面，使内胆与垫子套完全贴合好， 待其恢复原造型后把拉链拉上。在摆放垫子时，注意将有拉链或气孔的一侧靠后朝下，皮质靠垫的排气孔一侧朝后放置。

    在安全承受重量范围内，物件应轻拿轻放，以免瞬间受力过大造成家具损坏。
    • 家具安全承重标准如下：
    • 玻璃茶几、理石茶几承受重量：20公斤以下
    • 木面茶几承受重量：40公斤以下
    • 木面餐台承受重量：50公斤以下
    • 地柜、电视柜承受重量：60公斤以下
    • 厅柜、书柜、餐边柜、鞋柜层板承受重量：20公斤以下

    • 每两周进行一次表面的吸尘清理，清除织物缝隙及隐藏处的积尘。若沾染污渍，请用专业除渍产品立即清理，但在使用专业除渍产品前，在不明显处选小块位置先试用，确认该产品不会破坏面料颜色。织物面套的清洗要按照洗涤说明来操作，锐驰所有织物类产品均需干洗，请选择信誉好的专业清洗机构来清洗。
    • 每周至少拍打座垫及靠垫一次，使其充分透气，并保持内部填充物聚氨酯的回弹力及羽绒的蓬松度，以保证座垫及靠垫不会变形，延长其使用寿命。
    • 在多雨季节，为防止湿气的侵蚀，可在无雨时，用风筒吹沙发，及时除去沙发内的湿气。

    • 每两周进行一次表面的灰尘清理，用软布轻轻擦拭产品表面即可。若要对皮革产品进行吸尘，则必须使用软刷管口，以免擦破表面。若沾染污渍，请用软布沾少许皮革专用清理剂擦拭清理，请勿使用湿纸巾或湿布擦拭皮革产品。
    • 在夏季，皮革的空隙会吸收汗液，产生异味，要经常用潮干的软布擦拭，选用专业的清洁护理产品，以免发生皲裂。
    • 若不慎将蜡滴滴落到沙发上，在滴蜡处垫一块厚布，用熨斗熨一下，除去多余的蜡。如果口香糖掉落在沙发上，用保鲜膜裹着冰块置于粘口香糖处，使其变硬后轻轻刮掉。

    玻璃面家具沾染油污或其它污渍，可先用少许洗洁精去污，再用清水将洗洁精清洗干净，最后用干布擦干，注意避免让水流入柜内或抽屉内。雾化玻璃产品如沾染污渍，可用少许去污粉清洁。

    • 定期用潮干的软布顺着木纹方向擦拭木质家具产品。对于容易留下灰尘的角落处，应清洁彻底，因为灰尘长期不清除，会加速木质产品的老化速度。如家具沾染污渍，请立即清理。
    • 请勿使用含有汽油性质或酒精类等水溶溶剂擦拭家具。请勿使用湿纸巾或过于潮湿的棉布擦拭木质家具，以免湿纸巾中的酒精成分腐蚀家具面漆或造成面漆脱落。
    • 若较长时间未进行清洁，建议您使用专业的清洁剂。将清洁剂倒在柔软的棉布上进行清洁，然后把布翻转另一面进行擦拭，直到清洁剂被擦掉。

    • 铝合金可用微湿抹布擦拭，但不能用酸、碱性清洁剂，否则会腐蚀铝合金表面的保护层，发生氧化现象，影响性能及美观。同时应避免硬物碰撞或刮伤，因为磕碰的表面也会出现氧化现象。
    • 家具连接件在长时间使用后，个别部件的螺丝可能会有轻微松动，可用工具拧紧加固。
    • 滑轨一年至少要进行两次清洁保养，先清理滑轨处的积尘，再滴入少许润滑油，使其保持滑动顺畅。同时注意擦去溢出的润滑油，避免损坏家具表面。

    • 在清洗地毯时，可使用品质良好的干式泡沫清洁剂自行清理，或请专业的清洁公司来清洗。但切记，使用清洁剂前必须先将地毯吸干，免得使其背部组织受损，或腐蚀其内部结构。
    • 对于使用已久的纯毛地毯，最好调整一下位置，使其磨损均匀。如出现凹凸不平情况，请您轻轻拍打或用蒸汽熨斗、热毛巾敷熨。
    • 若不慎将液体洒落在地毯上，请您必须立即用吸水性好的纸巾或毛巾擦拭，尽可能的将污渍吸干，然后用海绵在污渍处沾上冷水再吸干。反复按照上述方法操作，直到将污渍吸附干净。这个方法也同样适用于洒落的红酒。但切忌用盐来擦拭，因为盐具有腐蚀性会损伤地毯。

    皮革会不会让人夏天感到很热，而冬天又非常冰冷？

    皮革会随着周围环境而改变属性。随着房间的温度和人体的体温，它会保持一个让你体感舒适的温度。

    我新买的皮革制品在表面有细小的纹路或印记，需要担心这些问题吗？

    遇到这样的问题，您不需要担心。因为皮革制品在表面有细小的印记，例如伤疤，虫咬，皱纹、条纹，以及由于天然牛皮每个部分的皮肤组织密度的不同产生的色差等都是原生皮的独特痕迹，是皮革的天然特点。CAMERICH的产品所使用的优质牛皮多选用圈养牛为主，此类皮质没有野牛的打斗伤痕，同时蚊虫叮咬留下的痕迹和病疾所致的疤痕也会比较少，并且每件产品都会尽量避免选取有色差的皮料，保证产品外观的精致和一致。

    为什么我的皮革沙发在使用了几年后就出现了松弛的现象？

    天然皮革所具有的弹性会对特定使用的区域产生定型。就像新购买的真皮鞋子要穿一阵子才能合脚是一样的道理。这种松弛不是工艺上的疏漏，而是皮革天然属性的一种体现。

    阳光照射会影响我沙发的颜色吗？

    无论是织物面料还是皮革，阳光直射都会影响其颜色。因为阳光直射会破坏面料及皮革的内部纤维组织，也会加速面料及皮革的褪色，经年累月后都有一定程度的颜色的变化。

    我买的沙发的织物面料洗涤后会变形和掉色么？

    不会。我们多年来重视对于沙发面料的选择和使用，我们使用的面料无论是在肌理、触感或是色彩的饱和度及色牢度上，都是获得全球经销商及客户所认可的。由于面料成分的特殊性，所有面料都要求干洗方式清洗，而且禁止漂白，这样可以保证面料组织不会被破坏，不会产生变形和掉色的现象，延长使用寿命。

    如何选择沙发织物类面套？

    • 棉麻面料：它的特点是柔软、透气、很贴近皮肤，花色最多样，自然环保。
    • 混纺 面料：它的特点是易洗免烫，色泽绚丽，结实耐用，富有弹性，耐磨性强。
    • 羊毛面料：它的特点是柔顺、柔软,较轻，吸湿和透气性好，但耐磨性较弱。
    • 皮布结合沙发：它的特点是在靠背扶手等易脏却又不易拆洗的地方用皮革制作，而其它与人体 亲密接触又易拆洗的地方用织物面料制作，风格新颖且耐脏，结合了皮革和织物面料的优点。

    如何选择实木类家具产品？

    纯实木家具（实木用量在90%－95％之间）和板木结合家具（实木用量在30％－90％之间)统称实木家具，现在市面上以板木结合的家具为主，纯实木的并不多见。
    • 纯实木家具：家具的所有用材都是实木，包括桌面、衣柜的门板、侧板等均采用实木制成，不使用其他任何形式的人造板。纯实木家具对工艺及材质要求很高，实木的选材、烘干、指接、拼缝等要求都很严格，如果哪一道工序把关不严，小则出现开裂、结合处松动等现象，大则整套家具变形，以至无法使用。
    • 板木结合家具：从外观上看是实木家具，木材的自然纹理、手感及色泽都和实木家具一模一样，但实际上是实木和人造板混用的家具，即侧板，顶、底、隔板等部件用薄木贴面的刨花板或中密度纤维板，门和抽屉则采用实木。这种工艺节约了木材，也降低了成本。 CAMERICH（锐驰）家具板件采用蜂窝结构技术，提高了板件的强度和刚性，具有良好的稳定性。我们使用低于国家甲醛含量标准（符合欧洲标准）的优质板材，板表面选用进口高级直纹橡木贴皮。表面油漆过程经过11道工序，并对每道工序都严格监管，最终的面漆经过测光仪器检测，将折光率严格控制在要求的范围内， 力求为消费者提供安全、环保的高品质产品。

    沙发中的填充物哪种好？

    聚氨酯填充：聚氨酯做沙发的填充物，具有较好的回弹性，柔软，透气，但是舒适度略显不足。
    羽绒填充：羽绒做沙发填充物，坐感柔软、舒适，在正常使用及保养情况下，不会产生变形现象，是一种成本高且慢回弹性的材质。 CAMERICH的设计师经过多年的设计研究,根据每款产品的特性,采用不同类型的填充方式, 使垫子的柔软舒适度达到最高标准,以满足不同的使用需求。

    如何拆装沙发面套？

    沙发面套的拆装是技术含量较高的工作，客户如需拆装，建议您联系当地CAMERICH（锐驰）授权专卖店，安排专人来为您操作。如需自行拆装，请严格按照以下步骤进行，以防错误的拆装方法损坏面套，给您带来不必要的损失。
    1. 拆面套：首先，将所有的坐垫、靠垫拿开，从沙发底部拉开面套的搭扣或拉链，用拉毛布（已配置）把搭扣粘上，以免面料挂住搭扣抽丝；然后，将面套由下往上缓缓地剥离，按照沙发的一边至另一边每次几厘米的步骤交替进行；最后，将面套完全从沙发上卸下。剥离过程切忌强拉硬拽，以免面套变形。
    2. 拆垫子套：首先，拉开座垫或靠垫的拉链，用力挤压内胆使其变小，然后从开口处轻轻抽出内胆，将面套完全拆下。
    3. 罩面套：首先要认清面套的正反面，从沙发最高处两侧开始套下，按照沙发的一边至另一边每次几厘米的步骤交替进行，特别注意是扶手及靠背的边角处要罩整齐，使布套贴紧沙发本体，抚平皱折；最后在沙发底部拉上拉链或按平搭扣。
    4. 罩垫子套：首先要认清垫子套的正反面；打开垫子套的拉链后，应注意内胆斜度方向与垫子套一致；其次用力挤压内胆使其变小，从拉链口处塞进，使内胆与面套基本贴合；最后双手拍打垫子各面，使内胆与垫子套完全贴合好，待其恢复原造型后把拉链拉上。在摆放垫子时，注意将有拉链或气孔的一侧靠后朝下，皮质靠垫的排气孔一侧朝后放置。
